第一：在第一次实现之前，先来安装依赖软件
```
➜  ~ bash temcheck.sh i
```
第二：查看其用法
```
➜  ~ bash temcheck.sh h
```
第三：程序的核心，那就是温度数据读取并实现桌面通知
```
➜  ~ bash temcheck.sh todo
```
